Liga / DesligaPressione para ligar ou desligar os LEDs.
MODE -Retrocede para o efeito anterior.
Brilho -Diminui a intensidade do brilho dos LEDs.
Diminui a sensibilidade no modo música.
Cadeado (Loop)Pressione por 5 segundos para realizar o emparelhamento com o controle remoto (a fita de LED piscará em verde 3 vezes).
Pressione novamente por 5 segundos para cancelar o emparelhamento (a fita de LED piscará em branco 3 vezes).
Reprodução de ciclo único: No modo de reprodução automática, pressione o botão "Engrenagem" uma vez e, em seguida, o botão "Cadeado" para capturar e reproduzir o efeito atual em loop.
Modo Música: 4 efeitos luminosos por tecla sincronizados com a música (microfone).
São 12 efeitos distribuidos entre as 3 teclas.
AUTOReproduz os diversos efeitos automaticamente.
Pressione 5 vezes para entrar no efeito de cores solidas RGB (facilita o teste da fita de LED).
"White” BrancoAtiva a cor branca fixa na fita de LED.
SPEED +Aumenta a velocidade dos efeitos de transição.
MODE +Avança para o próximo efeito.
Brilho +Aumenta a do brilho dos LEDs.
Aumenta a sensibilidade no modo música.
Configurar (Set)Ajusta a quantidade de LEDs existentes e a ordem correta das cores RGB para os botões de cores dos LEDs:
1. Pressione por 5 segundos esse botão para entrar no menu de configuração.
2. Pressione MODE+ e MODE- para ajustar a qtde de LEDs existentes.
3. Pressione os botões de cores para ajuste correto da ordem das cores RGB:
Pressione R até acender a cor vermelha.
Pressione G até acender a cor verde.
Pressione B até acender a cor azul.
4. Após terminar o ajuste, pressione novamente essa tecla para sair.
SPEED-Diminui a velocidade dos efeitos de transição.
R / G / B Seleção direta das cores (Vermelho, Verde e Azul).
Emparelhamento: Segure "Power" por 5s para parear (pisca verde 3x) ou desparear (pisca branco 3x).
Configuração RGB: No menu Set (segure 5s), use Mode+/- para pixels e R/G/B para calibrar a ordem das cores.
